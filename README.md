## 🛠 Инструменты и навыки (примененные в проектах)

•   **API Testing:** Работа с запросами GET, POST, DELETE в Postman, создание коллекций и окружений.
•   **Web Testing:** Использование Chrome DevTools (Network, Console, Elements) для анализа запросов и поиска багов.
•   **Тест-дизайн:** Проектирование тест-кейсов и чек-листов (позитивные/негативные сценарии, граничные значения).
•   **Баг-трекинг:** Составление детальных баг-репортов с четким описанием шагов, ожидаемых и фактических результатов.
